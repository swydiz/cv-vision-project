# Разработка и сравнительный анализ моделей компьютерного зрения для детектирования объектов в задачах навигации для слабовидящих

## Описание проекта

Данный проект выполнен в рамках учебной практики по дисциплине «Машинное обучение и компьютерное зрение».  
Цель проекта — разработка и сравнительный анализ современных моделей детектирования объектов для создания ассистивной системы, помогающей слабовидящим людям ориентироваться в пространстве.

## Основные возможности

- Детектирование объектов на изображениях с помощью 5 современных моделей
- Генерация пространственного описания сцены («слева», «справа», «по центру», «впереди» и т.д.)
- Речевое озвучивание описания на русском языке (pyttsx3)
- Визуализация результатов (аннотированные изображения)
- Сравнительный анализ моделей по точности, скорости и качеству локализации

## Использованные модели

- **YOLOv5**
- **YOLOv8n** (nano)
- **YOLOv8s** (small)
- **SSD (SSDLite320 MobileNetV3)**
- **Faster R-CNN (ResNet-50 FPN)**

## Как запустить

```bash
python main.py --model yolov8s --image path/to/image.jpg
Дополнительные параметры:

--folder path/to/folder — обработка всех изображений в папке
--no_speak — отключить голосовое озвучивание
```
